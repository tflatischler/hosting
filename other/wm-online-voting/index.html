Firebase einrichten

Die App braucht eine Firebase-Datenbank.

  1. Geh zu console.firebase.google.com
  2. Projekt hinzufugen → Name: wm-voting → Erstellen
  3. Build → Realtime Database → Datenbank erstellen → Im Testmodus starten
  4. Projekteinstellungen → Web-App hinzufugen (</>)
  5. Die firebaseConfig Werte in index.html einsetzen
